# QRcodeAPI
Web application interface for generating QR code
## Usage:
```text
http://localhost:8080/qrcode?url=http://google.com
http://localhost:8080/images/qrcode.png
```
## Server
### Linux
```shell
go mod tidy
go run main.go
```
